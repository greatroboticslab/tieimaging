# TIE Phase Reconstruction Project

## Overview
This project uses the Transport of Intensity Equation (TIE) to recover phase information from intensity images using MATLAB.

---

## Files in This Repo
- TIE_simple.m → main function
- run_TIE.m → runs the full process
- Tie Instructions → notes for how the code works

---

## How to Run the Code

1. Open MATLAB  
2. Go to this folder  
3. Run:
```
run_TIE
```

---

## What I Did (Experiment)

1. Took intensity images at different distances  
2. Used MATLAB to calculate intensity changes  
3. Applied TIE to recover phase
