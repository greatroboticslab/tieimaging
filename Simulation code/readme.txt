In this simulation, I found that using the raw intensity images acquired at z+ = +3 mm 
and z− = −3 mm relative to I0 produced more detailed features in the 3D reconstruction. 
Images with a "3" in the filename correspond to these datasets. MI05 remains the in-focus image (I0), 
while FFI05 is the corresponding flat-field image.

In this trial, we incorporate flat-field correction, which is performed by dividing
the pattern image by a uniformly illuminated reference (white) image. This process reduces 
the effects of uneven illumination, sensor variations, and background noise, allowing the 
LCD pattern to be observed more clearly.

We apply flat-fielding to improve reconstruction consistency and investigate the 
source of previously observed irregularities and non-uniform height variations.
While such artifacts are common in TIE phase reconstruction, we aim to determine whether 
they originate from the reconstruction process, the LCD pattern itself, or a damaged region
of the LCD screen that may contribute to the observed distortions.
